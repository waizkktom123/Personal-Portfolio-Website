A developer who refuses to build ordinary things.

I'm Muhammad Waiz Khan — a full-stack developer and software engineer based in Pakistan, engineering at the cutting edge of what's possible. I don't just write code; I architect experiences that challenge conventions.

My journey spans the full breadth of modern software — from pixel-perfect React & Next.js frontends to robust Node.js & Express backends, and from cross-platform Flutter mobile apps to enterprise .NET & C# systems.

I'm driven by a deep passion for clean architecture, performance, and developer experience. Whether it's crafting a real-time web app, building a Flutter UI, or designing a scalable REST API — I bring obsessive attention to detail to everything I create.

Currently exploring the intersection of modern web frameworks, mobile development, and cloud deployments. The future is written in code — and I'm here to write it.

05 — JOURNEY

The Origin Story

Every line of code has a story behind it.

2021
The First Line
Wrote my first HTML page and realised the web could be sculpted from text. Fell instantly in love with the idea of building things people can see and touch through a screen.
2022
JavaScript & React Deep Dive
Mastered vanilla JS and discovered React. Built component-based UIs, learned state management, and shipped first interactive web applications with real users.
2022–23
Full Stack Awakening
Added Node.js, Express, and PHP to the arsenal. Connected databases (MongoDB, MySQL), built REST APIs, and launched first production full-stack applications.
2023
Mobile & Enterprise
Ventured into Flutter and Dart for cross-platform mobile development. Simultaneously dived into .NET and C# for enterprise backend systems — becoming truly full-stack.
2024
Next.js & TypeScript
Levelled up frontend skills with Next.js, TypeScript, and Docker. Learned modern DevOps workflows, containerisation, and cloud deployment strategies.
2025+
The Future
Pushing deeper into scalable architectures, AI integrations, and production-grade applications. Building toward software that leaves a mark on the world.
